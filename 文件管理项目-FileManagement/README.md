# 文件系统管理项目

### 2253893 苗君文
## 项目简介

在内存中开辟一个空间作为文件存储器，在其上实现一个简单地文件系统。退出这个文件系统时，需要该文件系统的内容保存到磁盘上，以便下次可以将其恢复到内存中来。

## 实现功能

- 创建文件
- 打开文件
- 关闭文件
- 写文件
- 读文件
- 删除文件
- 格式化
- 创建子目录
- 删除子目录
- 显示目录
- 更改当前目录

## 运行方式

### 直接运行

   双击打开`./dist/FileSystem.exe`以运行程序。

### 编译运行

   安装python，可在conda环境下安装pyqt5。
   ```
   Python==3.11.5
   PyQt==5.15.7
   ```
   运行`main.py`文件即可运行。
